# pulsar_annual_timing_residuals

This repository cintains scripts to analyze pulsar timing files. To run using uv:

```bash
uv venv
. .venv/bin/activate
uv pip install -r requirements.txt
uv run vlbi_frozen_analysis.py
```

You also need to download the pulsar timing dataset from [Zenodo]([Zenodo](https://zenodo.org/records/16051178))
